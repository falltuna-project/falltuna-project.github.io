Day 7 氧化剂的pk

2020-08-11

问题: 用足量的KNO₃和O₂氧化等量的H₂SO₃溶液, 充分反应后哪一个溶液pH值更小?

解答:

1º H₂SO₃与O₂反应

2H₂SO₃ + O₂ → 4H⁺ + 2SO₄²⁻

2º H₂SO₃与KNO₃反应

3H₂SO₃ + 2NO₃⁻ → 2SO₄²⁻ + 2NO↑ 4H⁺ + H₂O

将两个反应方程中H₂SO₃系数调整一致

3H₂SO₃ + 1.5O₂ → 6H⁺ + 3SO₄²⁻
3H₂SO₃ + 2NO₃⁻ → 2SO₄²⁻ + 2NO↑ 4H⁺ + H₂O

显然, 等量的H₂SO₃与O₂反应时生成的H⁺物质的量, 大于与KNO₃反应时生成的H⁺物质的量, 且与KNO₃反应时生成H₂O, 稀释了H⁺

综上所述, 反应后, 与O₂反应组pH值更小.
